## JWT or JSON WEB TOKENS

- **pip install pyjwt**
  - PyJWT is a Python library which allows you to encode and decode JSON Web Tokens (JWT). 
  - JWT is an open, industry-standard (RFC 7519) for representing claims securely between two parties.
